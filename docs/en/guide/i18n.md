# i18n

## How to Use

```javascript
import Editor from "@hufe921/canvas-editor"

const instance = new Editor(container, <IElement[]>data, options)

// register
instance.register.langMap(locale: string, lang: ILang)

// set locale
instance.command.executeSetLocale(locale)
```
